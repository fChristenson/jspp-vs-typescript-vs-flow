Sublime Text JS++ Plugin
------------------------

This is a JS++ plugin for Sublime Text. This provides full support for JS++ in Sublime Text including syntax highlighting, build system integration with error locations, symbol navigation, documentation generator support, and so on.

This plugin has been tested with both Sublime Text 2 and Sublime Text 3.

Installation
---
1. Open Sublime Text
2. In the menu, go to Preferences -> Browse Packages... This should open your Sublime Text packages folder.
3. Create a folder named "JS++" in the Sublime Text packages folder.
4. Copy all the files into the "JS++" folder.
5. Restart Sublime Text.
6. Create a .jspp file, open it in Sublime Text, and select View -> Syntax -> JS++ -> JS++ from the menu.
